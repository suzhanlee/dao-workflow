# 회의록 템플릿

> 전문가 패널 기반 EBP(Evidence-Based Practice) 접근법으로 설계된 회의록 템플릿
>
> **Core Practice**: 섹션 분리 / 행위자 명시 / 시간 정보

---

| 항목 | 내용 |
|------|------|
| **회의 제목** | [회의 주제] |
| **일자** | YYYY-MM-DD |
| **시간** | 시작: HH:MM / 종료: HH:MM (소요: N분) |
